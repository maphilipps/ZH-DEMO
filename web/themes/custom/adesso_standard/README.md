# adesso standard Drupal theme

## Use Starterkit

```bash
ddev php web/core/scripts/drupal generate-theme test_theme --starterkit adesso_standard
```

See: https://www.drupal.org/docs/core-modules-and-themes/core-themes/starterkit-theme
