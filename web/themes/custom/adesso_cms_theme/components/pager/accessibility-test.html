Testing Instructions & Checklist

Keyboard Navigation Testing

  • Tab navigation through all links
  • Arrow key navigation within pagination
  • Enter/Space activation of links
  • Home/End key navigation
  • Focus indicators visible and clear
  • No keyboard traps

Screen Reader Testing

  • Navigation landmark announced
  • Descriptive context read
  • Current page clearly identified
  • Link labels descriptive
  • Disabled states announced
  • SVG icons properly hidden

Browser Dev Tools

  • Chrome Lighthouse (Accessibility)
  • Firefox Accessibility Inspector
  • Safari Web Inspector
  • axe DevTools extension
  • WAVE Web Accessibility Evaluator

Swiss Compliance

  • eCH-0059 focus indicators (3px+)
  • High contrast mode support
  • Municipal branding variations
  • Touch target sizes (44px+)
  • Multilingual label support
